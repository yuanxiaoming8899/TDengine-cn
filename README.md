<p dir="auto"><a href="https://tdengine.com" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">TDengine</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">是专为工业 4.0 和工业物联网构建的下一代数据历史数据库。</font><font style="vertical-align: inherit;">它可以实现每天 PB 级的实时数据摄取、存储、分析和分发，这些数据由数十亿个传感器和数据收集器生成。</font><font style="vertical-align: inherit;">TDengine 使大数据变得经济实惠且易于访问，数字化转型变得前所未有的简单。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">TDengine 与人工智能分析和其他第三方工具无缝集成，其开放生态系统为制造商提供整体解决方案，以推动 IIoT 时代的可行见解和运营效率。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">我们的使命是使工业数据系统民主化，让人们可以自由地使用他们的数据，但这对他们最有利。</font><font style="vertical-align: inherit;">我们决心将传统行业从阻碍数字化转型的遗留数据基础设施中解脱出来，走向一个打破孤岛、轻松集中和共享数据、让所有人都能享受到人工智能等革命性技术的好处的未来。工业部门之于大型科技公司。</font></font></p>

<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">目前，TDengine服务器支持在Linux/Windows/macOS系统上运行。</font><font style="vertical-align: inherit;">任何应用程序也可以选择taosAdapter提供的RESTful接口来连接taosd服务。</font><font style="vertical-align: inherit;">TDengine支持X64/ARM64 CPU，未来将支持MIPS64、Alpha64、ARM32、RISC-V等CPU架构。</font><font style="vertical-align: inherit;">目前我们不支持交叉编译环境的构建。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">您可以选择通过源码、</font></font><a href="https://docs.tdengine.com/get-started/docker/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">容器</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><a href="https://docs.tdengine.com/get-started/package/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">安装包</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">或</font></font><a href="https://docs.tdengine.com/deployment/k8s/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Kubernetes</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">安装。</font><font style="vertical-align: inherit;">本快速指南仅适用于从源安装。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">TDengine 提供了一些有用的工具，例如 taosBenchmark（被命名为 taosdemo）和 taosdump。</font><font style="vertical-align: inherit;">他们是 TDengine 的一部分。</font><font style="vertical-align: inherit;">TDengine编译默认不包含taosTools。</font><font style="vertical-align: inherit;">您可以使用</font></font><code>cmake .. -DBUILD_TOOLS=true</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">TDengine 来编译它们。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">要构建 TDengine，请</font><font style="vertical-align: inherit;">在项目目录中使用</font></font><a href="https://cmake.org/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">CMake 3.13.0 或更高版本。</font></font></a><font style="vertical-align: inherit;"></font></p>
